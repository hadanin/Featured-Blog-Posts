Having featured posts in a widget. Let's make WordPress life easier by marking any post as "Featured Post", and show 5 recent ones in the "Featured Posts Loader" widget anywhere on your website! So fast! So Easy!

== Description ==

Mark posts as featured ones, then place Feature Posts Widget anywhere in your website.

== Installation ==

1. Download the plugin file, unzip and place it in your wp-content/plugins/ folder. You can alternatively upload it via the WordPress plugin backend.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. "Featured Post" checkbox comes in Document sidebar.
